# Fake News Detector

## Overview
A simple AI-style fake news detection system using keyword-based analysis.

---

## Features
- Clean and preprocess news text
- Detect fake-style patterns
- Classify news as fake or real

---

## Files
- `preprocess.py` → cleans news text
- `detect.py` → detects fake news patterns

---

## Usage

```bash
python detect.py
```

---

## Example Input

```text
Breaking shocking secret revealed
```

---

## Example Output

```text
Fake News
```
